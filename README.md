# 🚗 Car Price Prediction using Machine Learning

## 📌 Project Overview
This project predicts the selling price of cars using Machine Learning techniques. It uses features like fuel type, kilometers driven, and transmission type.

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn

## 🤖 Model Used
- Linear Regression

## 📊 Features
- Present Price
- Kms Driven
- Fuel Type
- Selling Type
- Transmission
- Owner

## 📈 Output
- Predicted car prices
- Model evaluation (MAE & R2 Score)

## 🚀 How to Run
1. Install libraries:
   pip install pandas scikit-learn

2. Run the project:
   python analysis.py
